#### Application screen

![Screen](.github/screen.gif)

## Technologies

- [NodeJS 10.16.3](https://nodejs.org/)
- [ReactJS 16.9.0](https://reactjs.org/)

## Run

```bash
# After clone the project
$ cd bootcamp-gostack-desafio05

# Install packages
$ yarn install
# or
$ npm install

# Run the app
$ yarn start
```

The application will be available at: _http://localhost:3000/_
